# Rubric: audience-targeting

**What it measures:** Is there a clear, reachable audience that wants this content?

## Scoring Guide
| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No target audience defined |
| 0.3–0.6 | Vague audience ("business people," "entrepreneurs") |
| 0.6–0.8 | Specific audience with clear needs |
| 0.8–0.95 | Documented audience with evidence of demand |
| 0.95–1.0 | Highly specific audience + clear path to reach them |

## Key Questions
- Who exactly is this for?
- What problem does this solve for them?
- Where do they already hang out online?
