# Rubric: differentiation

**What it measures:** Is this genuinely different from what already exists?

## Scoring Guide
| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No differentiation — same as everyone else |
| 0.3–0.6 | Slight differentiation but not compelling |
| 0.6–0.8 | Clear angle that's different from competitors |
| 0.8–0.95 | Strong differentiation in voice, angle, or depth |
| 0.95–1.0 | Unmistakably unique — no one else is doing this |

## Key Questions
- Can you name 3 competitors and explain how this is different?
- Is the differentiation meaningful or superficial?
- Would the target audience recognize this as different?
