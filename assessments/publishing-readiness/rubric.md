# Rubric: publishing-readiness

**What it measures:** Is there a pipeline of ready-to-post content with a clear strategy?

## Scoring Guide
| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No content ready to post |
| 0.3–0.6 | Some drafts but no publishing strategy |
| 0.6–0.8 | Content bank exists + basic strategy |
| 0.8–0.95 | 15+ ready-to-post drafts + full publishing calendar |
| 0.95–1.0 | 30+ drafts, content calendar for first month, growth strategy defined |

## Key Questions
- How many posts are ready to go right now?
- Is there a publishing cadence defined?
- Is there a content calendar for the first month?
