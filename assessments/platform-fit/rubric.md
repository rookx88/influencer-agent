# Rubric: platform-fit

**What it measures:** Does the content format work well on LinkedIn + X?

## Scoring Guide
| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No platform strategy defined |
| 0.3–0.6 | Content exists but doesn't leverage platform strengths |
| 0.6–0.8 | Platform-appropriate formats identified |
| 0.8–0.95 | Content optimized for each platform's audience + format |
| 0.95–1.0 | Cross-platform strategy that works on both without dilution |

## Key Questions
- Is the format right for LinkedIn? For X?
- Does the content leverage each platform's strengths?
- Can the same idea work on both platforms with adaptation?
