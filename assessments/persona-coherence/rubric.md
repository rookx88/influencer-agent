# Rubric: persona-coherence

**What it measures:** Is the persona authentic, consistent, and differentiated?

## Scoring Guide
| Score | Meaning |
|-------|---------|
| 0.0–0.3 | No persona created |
| 0.3–0.6 | Generic persona, could be anyone |
| 0.6–0.8 | Clear persona with some differentiation |
| 0.8–0.95 | Authentic, consistent persona with clear POV |
| 0.95–1.0 | Compelling persona that stands out in the niche |

## Key Questions
- Does the persona have a clear point of view?
- Is it authentic or does it feel manufactured?
- Would you recognize this person's voice across different posts?
